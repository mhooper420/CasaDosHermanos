How to read the room panel
Dimension strings are wall-to-wall measurements copied exactly as hand-written on KB’s drawings (feet′-inches″). Values separated by / are sub-measurements along the same wall.
Source tags: “KB construction set (3.C1)” = measured off the real drawings · “drawing + estimate” = one axis measured, the other still estimated · “KB est.” = original schedule guess. The floor-plan shapes themselves are still the old distorted model — not to scale.
